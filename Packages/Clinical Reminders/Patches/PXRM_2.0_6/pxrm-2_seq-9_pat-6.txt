 General Overview:
 =================
 The main focus of this patch is the integration of Clinical
 Reminders with the new version of the Mental Health package,
 referred to as Mental Health Assistant 3 (MHA3). MHA3 is being
 released as patch YS*5.01*85 and is a required patch for PXRM*2*6.
 
 NOTE
 To use the full Mental health functionality in CPRS V27, YS BROKER1
 must be assigned as a secondary menu item (similar to OR CPRS GUI CHART)
 to all users of reminder dialogs. Also the YS_MHA.DLL file needs to be
 installed on all PC's in the Vist\Common Files folder.
 
 In addition to the integration with MHA3, this patch includes bug
 fixes, functionality enhancements required to support upcoming
 projects, and updated versions of a number of national reminders.
 
 This patch is distributed as a host file because in addition to the
 PXRM*2.0*6 build it contains GMTS*2.7*77, which distributes two new
 Health Summary components for MHA3: MHA Administration List and MHA
 Score.
 
 Complete details can be found in the Installation and Setup Guide;
 see the documentation section below for instructions on how to get
 it. Before the installation is done, the Clinical Reminder Manager
 at your site should read the Installation and Setup Guide to know
 what is being installed and be prepared if replacement items are
 needed during the installation of the reminders and dialogs.

 Installation:
 =============
 ***************DO NOT QUEUE THIS PATCH FOR INSTALLATION********
 
 Prior to installation please review APPENDIX C MHA3 & CR 6 Install Checklist
 of the Integration with New Mental Health Package Installation & Setup Guide.
 
 This patch can be loaded with users on the system. Installation should take
 less than 45 minutes.
 
 1. Use the 'Load a Distribution' option on the KIDS installation menu.
    When prompted to enter a host file type in PXRM_2_0_6.KID
 
 2. On the KIDS menu under the 'INSTALLATION' menu, use the following
    options, as desired:
        Print Transport Global
        Compare Transport Global to Current System
        Verify Checksums in Transport Global
        Backup a Transport Global
 
 3. On the KIDS menu under the 'INSTALLATION' menu, use the following
    option to install the patch:
        Install Package(s) (PXRM*2.0*6)
 
 4. When prompted "Want KIDS to INHIBIT LOGONs during the install? NO//,"
    respond 'NO'.
 
 5. When prompted "Want to DISABLE Scheduled Options, Menu Options,
    and Protocols? NO//," respond 'NO'. Note - all the appropriate
    options and protocols will be automatically disabled and renabled.
 
 6. When prompted "Delay Install (Minutes):  (0-60): 0//," respond '0'.
    Note - during the installation of the reminders and dialogs you
    may be ask for replacement items; it is advisable to have your
    Clinical Reminder Manager on hand to answer these questions.
 
 7. The init routines PXRMP6I* may be deleted once the installation has
    completed.
