Description:
============

 IB*2.0*426 is being released to support the Treating Specialty changes 
 released in patch DG*5.3*819.
  
 Treating Specialties in the SPECIALTY (#42.4) file are being updated by 
 patch DG*5.3*819.  Patch IB*2.0*426 updates Integrated Billing to 
 properly handle the billing of the new Treating Specialties.
  
 *************************************************************************
 **                                                                     **
 ** NOTE: This patch can be installed at any time as the three          **
 ** Treating Specialty codes added with this patch will not be used in  **
 ** billing until they are selectable in Registration (which will       **
 ** happen with the install of DG*5.3*819).  That is the only change    **
 ** with this patch.                                                    **
 **                                                                     **
 *************************************************************************

 REMEDY OVERVIEW:
 ================
 1) HD0000000347629    Need Updated Treating Specialty Code
  
  Problem: 
  --------
  Treating Specialties in the SPECIALTY (#42.4) file are being updated by 
  patch DG*5.3*819 and require updates to the billing software. 
    
  Resolution:
  -----------
  There will be 3 new LTC (Long Term Care) Treating Specialties. The 
  following details the billing set up for these new Treating 
  Specialties.
  
  
 New Treating Specialties and Long Term Care
 -------------------------------------------
 The following new Treating Specialties have been identified as Long Term
 Care Treating Specialties and are assigned to the following Long Term
 Care Copayment.
  
   Code  Treating Specialty                Long Term Care Copay
   ----  -------------------------         --------------------
   1K    PSYCH RESID REHAB PROG            LTC INPT DOM
               
   1L    PTSD RESID REHAB PROG             LTC INPT DOM
              
   1M    SUBSTANCE ABUSE RESID PROG        LTC INPT DOM
