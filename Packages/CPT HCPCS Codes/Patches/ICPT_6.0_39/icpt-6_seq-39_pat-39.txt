Subject: Code Set FY08 2nd Qtr Update

Category: 
  - Informational

Description:
============

  This patch is an informational patch that contains CPT/HCPCS changes for
  FY 2008 2nd quarter, effective Jan 1, 2008, and adds the RT CPT modifier 
  for CPT Code 34812.  This patch contains no routines, data, or data
  dictionaries.  The data for this patch is exported with the corresponding 
  Lexicon data in patch LEX*2.0*54.  Patch LEX*2.0*54 requires that patch 
  LEX*2.0*52 be previously installed.
     
  The data in LEX*2.0*54 affects the following CPT/HCPCS files: 
 
        CPT/HCPCS Procedure      #81         ^ICPT
        CPT Category             #81.1       ^DIC(81.1)
        CPT Modifier             #81.3       ^DIC(81.3)
 
  REMEDY TICKET
  =============
  
    RT Modifier for CPT Code 34812     HD0000000 222701  Minneapolis, MN
