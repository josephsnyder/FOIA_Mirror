Subject: Code Set FY10 2nd Qtr Update

Category: 
  - Informational

Description:
============

  This patch is an informational patch that contains CPT changes for 
  FY 2010 2nd Quarter and applies fixes for 4 Remedy Tickets.  This
  patch contains no routines, data, or data dictionaries.  The data
  for this patch is exported with the corresponding Lexicon data in
  patch LEX*2.0*68.  Patch LEX*2.0*68 requires that patches LEX*2.0*62,
  LEX*2.0*55, LEX*2.0*67 and ICD*18.0*45 be previously installed.
   
  This patch writes to the following files:
 
       CPT                       #81       ^ICPT(   
       CPT CATEGORY              #81.1     ^DIC(81.1,
       CPT COPYRIGHT             #81.2     ^DIC(81.2,
       CPT MODIFIER              #81.3     ^DIC(81.3,
