Subject: REMOVE TEMPORARY MENU OPTIONS

 with patch PSO*7.0*288.

 REMEDY OVERVIEW:
 ================
 HD0000000257416     Need to remove temporary menu options
  
   
 Problem: 
 --------
 This patch will remove the temporary menu options set up with patch
 PSO*7.0*288.  The menu options exist under the Maintenance (Outpatient
 Pharmacy) menu [PSO MAINTENANCE]. Menu option Pharmacy Patient Non-VA Meds
 Report/Clean-up [PSOZ NON-VA MEDS REP/CLEAN] has two other menu options to
 create the report and allow the user to clean up the erroneous data.  
 These two options are Pharmacy Patient Erroneous Non-VA Meds Report [PSO
 NON-VA MEDS REPORT] and Clean up Pharmacy Patient Non-VA Meds data [PSO
 NON-VA MEDS CLEAN-UP].
  
  
 Resolution:
 -----------
 This patch will automatically remove the menu options created with
 patch PSO*7.0*288.
  
 **Do not install this patch until after finishing the clean up of any
 erroneous Non-VA Meds records.**
  
 ***The updating of the user manual due to the removal of these temporary 
 menu options will take place when patch PSO*7*320 is released in the 
 future.***
  
  
        ================INSTALLATION INSTRUCTIONS =================
  
 It is not necessary to disable any options.  You may install 
 these routines at any time without interruption to users.  It should
 take less than 2 minutes to install this patch.
  
    1. Choose the PackMan message containing this patch and invoke the
       INSTALL/CHECK MESSAGE PackMan option.  
   
    2. From the Kernel Installation & Distribution System menu, select 
       the Installation menu.  
   
    3. From this menu, you may select to use the following options: 
       (when prompted for INSTALL NAME, enter PSO*7.0*311)
   
         a.  Verify Checksums in Transport Global - This option will allow
             you to ensure the integrity of the routines that are in the
             transport global.
         b.  Print Transport Global - This option will allow you to view
             the components of the KIDS build.
         c.  Compare Transport Global to Current System - This option will
             allow you to view all changes that will be made when this patch
             is installed.  It compares all components of this patch
             (routines, DD's, templates, etc.).
         d.  Backup a Transport Global - This option will create a backup
             message of any routines exported with this patch. It will not
             backup any other changes such as DD's or templates.
   
    4. Use the Install Package(s) option and select the package PSO*7.0*311.
   
    5. Want KIDS to Rebuild Menu Trees Upon Completion of Install? 
       NO//respond NO.
  
    6. When Prompted "Want KIDS to INHIBIT LOGONs during the install? 
       NO//" respond NO.  
   
    7. When Prompted "Want to DISABLE Scheduled Options, Menu Options, and 
       Protocols? NO//" respond NO.
