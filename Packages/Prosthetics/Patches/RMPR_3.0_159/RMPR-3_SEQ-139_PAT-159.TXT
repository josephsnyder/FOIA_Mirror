Subject: NEW NPPD LINE;HOME OXYGEN REPORT PRINT UNER LINUX;SSN ON HOME OXYGEN LETTERS

Category: 
  - Routine

Description:
============

 This patch addresses the following issues:
  
 1. New Prosthetics line number R99 B - NONRESPONSE will be added to the 
 system.
  
 2. Sites that are using the LINUX operating system are receiving an error 
 when attempting to print the Home Oxygen report.
  
 3. When Home Oxygen letters are being produced, the system is printing the
 patients full SSN on them.

 Remedy Overview:
 ================ 
 1) HD0000000393954 New Prosthetics line number: R99 B  NONRESPONSE
  
 Problem:
 --------
 Add new NPPD line number 'R99 B   NONRESPONSE'.
  
   
 Resolution:
 -----------
 Modify routines RMPRN62 and RMPRN72 for new NPPD line number.
  
 Routines: RMPRN62, RMPRN72
  
  
  
 2) HD0000000390723 Cannot print home oxygen report
  
 Problem:
 --------
 Site using LINUX operating system reported error when report
 queued.
  
  
 Resolution:
 -----------
 Found that Monthly Home Oxygen Billing report used ZTIO=IO and
 should have been ZTIO=ION.  Also, corrected same problem for reports:
 RMPR LAB STOCK PENDING
 RMPR INV PRINT LAB ITEM
  
 Routines: RMPORMB, RMPR29LP, RMPR5N4
  
  
 3) HD0000000401779 Entire SSN appears on Home Oxygen letter
  
 Problem:
 --------
 Site using Home Oxygen option [Generate Letters] reports that the entire 
 patient social security number prints on letter.
  
  
 Resolution:
 -----------
 Modified letter printing to NOT print patient social security number at 
 all when print Home Oxygen letters.
  
 Routines: RMPOLET2, RMPOLZB, RMPOLF0, RMPOLF0A

        ================INSTALLATION INSTRUCTIONS ================= 
  
 You may install these routines at any time without interruption to 
 users. It should take less than 2 minutes to install this patch. 
  
    1. Choose the PackMan message containing this patch and invoke the
       INSTALL/CHECK MESSAGE PackMan option.  
   
    2. From the Kernel Installation & Distribution System menu, select 
       the Installation menu.  
   
    3. From this menu, you may select to use the following options: 
       (when prompted for INSTALL NAME, enter RMPR*3.0*159)
   
         a.  Verify Checksums in Transport Global - This option allows
             you to ensure the integrity of the routines that are in the
             transport global.
         b.  Print Transport Global - This option allows you to view
             the components of the KIDS build.
         c.  Compare Transport Global to Current System - This option 
             allows you to view all changes that are made when this 
             patch is installed.  It compares all components of this patch
             (routines, DD's, templates, etc.).
         d.  Backup a Transport Global - This option creates a backup
             message of any routines exported with this patch. It will not
             backup any other changes such as DD's or templates.
   
    4. Use the Install Package(s) option and select the package
       RMPR*3.0*159.
   
    5. When Prompted "Want KIDS to INHIBIT LOGONs during the install? 
       NO//" respond NO.  
   
    6. When Prompted "Want to DISABLE Scheduled Options, Menu Options, 
       and Protocols? NO//" respond YES. When prompted to select the 
       options you would like to place out of order, enter the following:
                                                                  
       'RMPO-RPT-HOMONBIL'         Monthly Home Oxygen Billing
       'RMPR LAB STOCK PENDING'    Lab Stock Issue Pending Completion
       'RMPR INV PRINT LAB ITEM'   Print Employee Lab Issue Statistics
       'RMPO LIST/PRINT MANAGER'   Generate Letters
