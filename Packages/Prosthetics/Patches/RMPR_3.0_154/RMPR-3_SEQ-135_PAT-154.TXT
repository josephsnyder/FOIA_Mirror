Subject: REMOVE PCE INTERFACE ROUTINES;HOME OXYGEN BILLING NAMES;RMPR EDIT LOCATION ERROR

Category: 
  - Routine

Description:
============

 This patch will address the following 3 issues:
  
 1) Remove the routines related to the PCE interface which was removed by 
 patch RMPR*3*145
  
 2) Modify the Home Oxygen billing, both Purchase Card & 1358 based, to 
 not attach patient names or partial social security numbers to orders 
 created.
  
 3) Correct the error in option RMPR INV EDIT LOCATION when the user 
 enters '??' at the Location Name prompt.

 Remedy Overview:
 ================ 
 1) Vista Prosthetics, remove routines RMPRPCEA and RMPRPCEB
  
 Problem:
 --------
 Remove the routines related to the PCE interface that was removed in 
 patch RMPR*3*145.
  
   
 Resolution:
 -----------
 Delete routines RMPRPCEA and RMPRPCEA used during the daily run to 
 create PCE encounters for Prosthetics orders and issues.  Also, modify
 issue edit/delete routines calling these deleted routines.
  
 Routines: RMPRPCEA, RMPRPCEB, RMPRED4, RMPRPIYF, RMPRPS34
  
  
  
 2) HD0000000321817 Home Oxygen Purchase Order in IFCAP
  
 Problem:
 --------
 Modify the Home Oxygen billing, both Purchase Card or 1358 based, to
 not attach patient names or partial social security numbers to orders 
 created.
  
  
 Resolution:
 -----------
 Modify routine RMPOPST1 to summarize the patients on a Purchase Card 
 Home Oxygen order and modify the patient name/social security number
 sent to a 1358 request to be an anonymous patient identifier only.
  
 Routines: RMPOPST1 
  
  
  
 3) HD0000000303222 No line tag called ELQQM in routine RMPRPIY6
  
 Problem:
 --------
 When using option RMPR INV EDIT LOCATION an error will occur when 
 entering a '??' at the Location Name prompt.
  
 Resolution:
 -----------
 Modify routine RMPRPIYN to call the correct tag within the same routine 
 when user enters a '??' at the Location Name prompt.
  
 Routines: RMPRPIYN

 You may install these routines at any time without interruption to

       INSTALL/CHECK MESSAGE PackMan option.

       the Installation menu.  
   
    3. From this menu, you may select to use the following options: 
       (when prompted for INSTALL NAME, enter RMPR*3.0*154)
   
         a.  Verify Checksums in Transport Global - This option allows

             transport global.
         b.  Print Transport Global - This option allows you to view
             the components of the KIDS build.
         c.  Compare Transport Global to Current System - This option 
             allows you to view all changes that are made when this 
             patch is installed.  It compares all components of this patch
             (routines, DD's, templates, etc.).
         d.  Backup a Transport Global - This option creates a backup
             message of any routines exported with this patch. It will not

       RMPR*3.0*154.
