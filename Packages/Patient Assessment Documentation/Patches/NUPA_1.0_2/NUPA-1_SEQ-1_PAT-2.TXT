 Description:
 ===========
  
 This patch is an informational patch and contains no routines, data, or 
 data dictionaries.  The Patient Assessment Documentation Package was 
 released April 12, 2012 and has been installed at all of the 126 VistA 
 Sites.  Sites may continue to utilize the current version of this package,
 however this patch is the official notice that the Patient Assessment 
 Documentation Package is no longer supported by VA local or the national 
 support teams.
