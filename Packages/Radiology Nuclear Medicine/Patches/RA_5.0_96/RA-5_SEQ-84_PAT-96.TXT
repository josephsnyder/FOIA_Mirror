Category: 
  - Routine
  - Enhancement (Mandatory)

Description:
============

 This patch will address the following issue:
       
 The Radiology Program Office distributes a yearly revised listing
 of Radiology Current Procedural Terminology (CPT) codes.  These revisions
 will be applied to the CPT by Procedure Type (#73.2) file and will be used
 by the sites for scheduling exams.

 REMEDY OVERVIEW:
 ================     
 HD0000000263980 - Maintenance of file 73.2
       
 PROBLEM
 -------
 File (#73.2),RADIOLOGY CPT BY PROCEDURE TYPE, must be updated
 with the yearly updates provided by the Radiology Program
 Office. 
        
 RESOLUTION 
 ----------     
 This patch will update the RADIOLOGY CPT BY PROCEDURE TYPE (#73.2)
 file for distribution to the sites.  Along with several cosmetic changes,
 new CPT codes will be added and the Wait Time Procedure Type field will be
 updated on several CPT codes. 
  
 Also a post-init routine will be executed to delete CPT code 90659 from
 the file as it is no longer required.
