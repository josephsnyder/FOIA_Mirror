Subject: Ability to add OC type Training Facility in New Person File

Category: 
  - Routine
  - Data Dictionary

Description:
============

 This patch corrects the following issue:
  
 When making a selection for the VHA TRAINING FACILITY (#12.4) field of
 the NEW PERSON (#200) file, the screen was screening out stations whose
 FACILITY TYPE (#13) field in the INSTITUTION (#4) file was "OC", meaning
 "OUTPATIENT CLINIC (INDEPENDENT)".
  
 Please see the example below:
  
 Before:
 [snip]
    DURHAM VAMC                 NC  VAMC      558  
    EAST ORANGE-VA NEW JERSEY HCS     NJ  VAMC  561  
    EASTERN COLORADO HCS        CO  VAMC      554  
    ERIE VAMC                   PA  VAMC      562  
    FARGO VAMROC                ND  M&ROC     437  
 [snip]
  
 After:
 [snip]
    DURHAM VAMC                 NC  VAMC      558  
    EAST ORANGE-VA NEW JERSEY HCS     NJ  VAMC  561  
    EASTERN COLORADO HCS        CO  VAMC      554  
    EL PASO VA HCS              TX  OC        756  
    ERIE VAMC                   PA  VAMC      562  
    FARGO VAMROC                ND  M&ROC     437  
 [snip]
  
 The screen explanation for the field has been appropriately updated.

 Installation Instructions: 
 ==========================
 This patch can be installed with VISTA users online.  However, it is 
 recommended that it be Queued for a time of generally least activity.  
  
 TaskMan does not need to be STOPPED or placed in a WAIT state and 
 installation should take less than a minute.
  
      1.  Use the 'INSTALL/CHECK MESSAGE' option on the PackMan menu.  
 This option will load the KIDS package onto your system.  
  
      2.  The patch has now been loaded into a transport global on your 
 system. You now need to use KIDS to install the transport global.  
   
      3.  On the KIDS menu, under the 'Installation' menu, use the 
 following options: 
             Print Transport Global 
             Compare Transport Global to Current System 
             Verify Checksums in Transport Global 
             Backup a Transport Global 
  
      4.  On the KIDS menu, under the 'Installation' menu, use the 
 following option: 
             Select Installation Option: Install Package(s) 
             Select INSTALL NAME: XU*8.0*401
                                  ==========
         Answer 'NO' to 'Want KIDS to INHIBIT LOGONs during the install?
         Answer 'NO' to 'Want to DISABLE Scheduled Options, Menu Options,
 and Protocols?
