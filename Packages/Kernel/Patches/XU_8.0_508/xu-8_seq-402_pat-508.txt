                         XU*8*508
  
 This patch does following items:
  
 - Fixes the <UNDEFINED> error for the 'Option Access By User' [XUOPTWHO]
   when choosing to list users for the option SYSTEM COMMAND OPTIONS
   [XUCOMMAND].
  
 - Deletes the option 'Program Integrity Checker' [XUINTEG] at Sites.
   This option is no longer used, and is replaced by the option Compare 
   local/national checksums report [XU CHECKSUM REPORT].

 Installation Instructions
 =========================
  
 1. Users ARE allowed to be on the system during the installation. 
    However, it is recommended that the patch be installed during
    non-peak time. It will take about a minute to install the
    patch, but you may queue the installation if you wish.
  
 2. You DO NOT need to stop TaskMan or the background filers.
  
 3. Use the 'INSTALL/CHECK MESSAGE' option on the PackMan menu. This      
    option will load the KIDS package onto your system.
  
 4. The patch has now been loaded into a Transport global on your system. 
    You now need to use KIDS to install the Transport global. On the KIDS
    menu, under the 'Installation' menu, use the following options:
  
                 Verify Checksums in Transport Global
                 Print Transport Global
                 Compare Transport Global to Current System
                 Backup a Transport Global
  
    On the KIDS menu, under the 'Installation' menu, use the following
    option:
                 Install Package(s)  
                 Select INSTALL NAME:XU*8.0*508
                                     ========== 
  
   Accept the defaults for the following install questions:  
  
   Want KIDS to Rebuild Menu Trees Upon Completion of Install? NO//
   Want to DISABLE Scheduled Options, Menu Options, and Protocols? NO// 
   Want KIDS to INHIBIT LOGONs during the install?  NO//
  
 5. Delete the routine XUGET if you wish (code for option 'Program 
    Integrity Checker' [XUINTEG]):
  
   Programmer Options menu --> Routine Tools Menu --> Delete Routines
   option
