 REMEDY OVERVIEW:
 ================
 1).  HD0000000219890 - ECX PROS COST report - not showing duplicate 660
                        records
  
 Problem:
 --------
 The REMEDY ticket is needed for the "Prosthetic Extract Unusual Cost Report 
 [ECX PRO COST]" option's failure to capture all pertinent data.
  
 Resolution:
 -----------
 Routines ECXUPRO and ECXUPRO1 were modified to capture all pertinent data for 
 the "Prosthetic Extract Unusual Cost Report [ECX PRO COST]".
  
 REMEDY OVERVIEW:
 ================
 2). HD0000000225935 -Undefined error during Pharmacy Extract.
  
 Problem:
 --------
 Pharmacy Extracts Incomplete Feeder Key Report [ECX PHA FKEY] getting an 
 UNDEFINED ERROR in ECXDRUG2.
  
  
 Resolution:
 -----------
 Routine ECXDRUG2 was modified to prevent the UNDEFINED ERROR in Pharmacy 
 Extracts Incomplete Feeder Key Report [ECX PHA FKEY].
  
  
 REMEDY OVERVIEW:
 ================
 3).  HD0000000225781 - Pt Holding Times are negative numbers
  
 Problem:
 --------
 Surgery Unusual Volume Report listing zeros for PT Holding time column.
  
  
 Resolution:
 -----------
 Routine ECXUSUR1 was modified to correct Surgery Unusual Volume Report listing 
 incorrect zeros for PT Holding time column. 
  
 REMEDY OVERVIEW:
 ================
 4).   HD0000000226317 - (ECXNUT) not picking up the patient division
                         consistently.
  
 Problem:
 --------
 The DSS Nutrition Extract (ECXNUT) is not picking up the patient division 
 consistently. When we run the Extract Audit Report (ECX NUT SOURCE AUDIT) 
 we get many entries with Patient Division Unknown. 
  
  
 Resolution:
 -----------
 Routine ECXANUT was modified to correct the Patient Division from 
 displaying Unknown on the detailed Nutrition Audit Report.

           allow you to view all changes that will be made when the patch
           is installed. It compares all components of the patch (routines,
           DDs, templates, etc.)
        c. Verify Checksums in Transport Global - this option will ensure
           the integrity of the routines that are in the transport global.
        d. Print Transport Global - this option will allow you to view the
           components of the KIDS build.

      5. When prompted "Want KIDS to INHIBIT LOGONs during the install?
         YES//" respond NO.
