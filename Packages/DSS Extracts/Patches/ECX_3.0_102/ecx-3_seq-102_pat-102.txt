Description:
============

 A new routine was created to extract records from the VistA Blood 
 Establishment Computer Software (VBECS), Blood Bank Modernization project
 that is scheduled to be deployed in FY 08 to all medical centers.
  
 - The Purge Data from Extract Files [ECXPURG] option has been modified 
 to allow purging of the VBECS intermediate file.
  
 - The Laboratory Blood Bank (LBB) Extract Audit [ECX LBB PRE-EXTRACT] 
 option has been modified to incorporate VBECS records.

            NAME, enter ECX*3.0*102):
       
     a.  Backup a Transport Global - This option will create a backup
         message of any routines exported with this patch. It will not
         backup any other changes such as DD's or templates.
     b.  Compare Transport Global to Current System - This option will
         allow you to view all changes that will be made when this patch

         (routines, DD's, templates, etc.).

         transport global.
