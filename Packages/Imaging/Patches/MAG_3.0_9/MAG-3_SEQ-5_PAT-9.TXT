Description: 
============

 VistA Imaging's autorouting and on-demand routing software is released as 
 patch MAG*3.0*9.  The Routing software is an addition to VistA Imaging 3.0
 software and is used to automatically and/or manually transmit image files
 over a Wide Area Network (WAN) for remote diagnosis or teleradiology.  
 This software offers many site configurable options to meet the demands of
 the individual site's requirements for remote viewing of images on
 VistARad workstations. For detail information on this patch review the
 MAG3_0P9 Patch description posted on the Imaging web page at 
 http:/vaww.va.gov/Imaging/3.0patches.htm.
  
 This patch requires Imaging's patch MAG*3.0*16 prior to installation. 
 Imaging sites that are testing consolidation software should not install 
 this patch.
  
 The patch consists of the following files.
  
 readme.doc - Installation steps and last-minute information about the 
 routing software.
  
 Mag3_0P9.kid - KIDS package to be installed on VISTA hospital database.
  
 MAGDCM.RTN - Routine save package to be installed on the DICOM Gateways.
  
 Routing User Guide.pdf - Configuration and end user documentation for 
 routing.
  
 DATA_CR.DIC, DATA_MRI.DIC, DATAGECT.DIC, DATAMISC.DIC, ELEMENT.DIC, 
 HL7.DIC, SCP_LIST.DIC, TEMPLATE.DIC, UID.DIC - Files containing 
 definitions related to the DICOM standard (to be installed only on the 
 DICOM Gateways).
  
  
 To receive this patch, a site must have their Site Agreement on file with
 and approved by the VistA Imaging Group.
  
 Once the required documents are received and approved, software will be
 available on the Imaging FTP directory under your site's folder. For
 information concerning the status of a site, contact Katy Trombetta via
 phone (310-734-0357) or email (Kathy.Trombetta@med.va.gov).
