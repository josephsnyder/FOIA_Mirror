Subject: VISTARAD 3-D INTEGRATION AND 508 SUPPORT

Category: 
  - Other
  - Routine

Description:
============

 This document describes Patch 76, the VistARad 3-D Integration and 508
 Support patch. This patch incorporates:
  
 * Support for the Voxar 3-D Workstation application.
 * New functionality for Section 508 compliance.
 * Fixes for miscellaneous issues.
  
 NOTE  With the release of Patch 76, VistARad Patches 32 or earlier will no 
       longer be supported. 
  
 NOTE  All sites running VistA Imaging 3.0 must install the KIDS portion of
       this patch. At sites that have implemented VistARad, installation of 
       both the KIDS and the client software is required; installation of the
       client software may be phased in according to each site s needs. 
    
 Associated Patches
 ==================
   
 Before Patch 76 is installed, the following patches must be installed:
 MAG*3.0*65
  
 NOTE  Patch 54, while not required for the installation of Patch 76, 
       does affect Patch 76 functionality.  See the Patch 76 Changes 
       section for details. 
  
 For a list of all released Imaging (MAG) patches, refer to the 
 patch_list.txt file available on the Imaging FTP server (go to 
 ftp://ftp.imaging.med.va.gov, then navigate to the 
 \Docs\Imaging_Docs_Latest folder).

 Security Keys:
 =============
  
 The following new security keys are included in this patch:
  
 MAGJ SYSTEM MANAGER   Allows access to Voxar-related settings in the VistARad 
   Settings dialog.  Should only be assigned to VistARad administrators.
  
 MAGJ STORE IMAGES   Allows VistARad users to save Voxar images as secondary 
   captures to VistA (additional configuration and Patch 54 required, see the 
   Installation section in this document for details).
  
 MAGJ VOXAR EXPORTCAPTURE   Allows VistARad users to export images using Voxar 
   (Enables the three Export-related buttons in the Voxar Reading manager 
   window; for information about exporting images, see the Voxar 3D 
   documentation.)
  
 MAGJ VOXAR COPYIMAGE   Allows VistARad users to copy images using Voxar 
   (Enables the Copy to Clipboard button in the Voxar Reading manager window; 
   for information about copying images, see the Voxar 3D documentation.)
  
 MAGJ VOXAR PRINTCOMPOSER   Allows VistARad users to print images using Voxar 
   (Enables the Print Composer button in the Voxar Reading manager window; 
   for information about printing images, see the Voxar 3D documentation.)
