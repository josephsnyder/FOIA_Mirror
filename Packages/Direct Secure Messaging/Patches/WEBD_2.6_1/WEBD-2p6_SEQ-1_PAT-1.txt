Subject: Direct Secure Messaging Phase 3 Version 2.6.0

Category: 
  - Informational

Description:
============

 Direct Secure Messaging (DSM) Release 2.6.0 implements upgrades of 
 commercial-off-the-shelf components Apache Web Server, Postfix, PHP as 
 well as the decommissioning of Windows Server 2008. These changes are Work
 Items that were recorded and tracked in the Direct Secure Messaging area 
 of Rational Team Concert, the VA-prescribed tool used for tracking 
 requirements and associated test plans. This release contains the Work 
 Items completed during a production deployment. 
  
 Supporting technical documentation including the DSM System Design 
 Document, User Guide, Production Operation Manual (POM), and
 Install/Rollback Guide were updated to reflect the above-mentioned 
 application enhancements.  
  
 Test sites: N/A
  
 The upgraded software was deployed to a centralized system at AITC on
 December 13th, 2019.
  
 External Partner Test Sites: 
  
 Partner testing will be performed and verified to be working as expected 
 as part of DSMEP3 Release 2.6.0.
  
 Downstream Dependent System Testing:
  
 DSMEP3 Release 2.6.0 test plan's coverage includes testing efforts with
 eHealth Exchange, VLER DAS, Master Veteran Index and 
 Veterans Authorizations and Preference.
  
 ServiceNow tickets: None
